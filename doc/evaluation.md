# Evaluation

## Evaluation Focus
The evaluation extends beyond simple detection. It measures whether the organisation would retain operational capability during a disruptive event.

## Recommended Evaluation Questions
1. How long did it take for Wazuh to surface the first meaningful alert?
2. Was the alert specific enough to support investigation?
3. Did the API remain available throughout the test window?
4. Did latency increase materially during the simulation?
5. What does the evidence suggest about the organisation's resilience posture?

## Example Discussion Structure
### Detection Effectiveness
Discuss alert timeliness, fidelity, and visibility.

### Service Continuity
Discuss uptime, user-facing availability, and performance.

### Strategic Relevance
Explain how technical evidence maps to cyber resilience, incident response maturity, and business continuity.
